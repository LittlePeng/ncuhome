[czx] 对 [胡泰斌经理] >> 学生票到庐山多少钱？

[胡泰斌经理] 对 [czx] >> 带学生证的话去庐山优惠35。

[boud] 对 [胡泰斌经理] >> 能说说具体的安排吗?    

[胡泰斌经理] 对 [boud] >> 具体安排还是请你到我们办公室来咨询好吗？庐山的话，五一期间每天都有专线。

 

[boud] 对 [胡泰斌经理] >> 您觉得我们学生去哪里旅游更好、更划算？ 

[胡泰斌经理] 对 [boud] >> 主要是省内的更划算啊，九江，庐山。

[boud] 对 [胡泰斌经理] >> 我们班上要去庐山游玩，你觉得多人去旅游，大概花费为多少？平均的。

[boud] 对 [胡泰斌经理] >> 但是我们平时没有时间啊，双休日来得急游玩庐山的景点吗？

[胡泰斌经理] 对 [boud] >> 可以啊，庐山一般都是一日游或者两日游。

[胡泰斌经理] 对 [boud] >> 两日游当然更过瘾。

[boud] 对 [胡泰斌经理] >> 那双休日去游玩，会不会很紧啊？我们要学习，怕紧了的话，没有时间休息啊。

[胡泰斌经理] 对 [boud] >> 不会啊，事先计划好就不会很匆忙了。

[boud] 对 [胡泰斌经理] >> 那么那里的住宿情况怎么样呢？

[胡泰斌经理] 对 [boud] >> 住宿可以放心的，庐山作为国内著名景点已经开发非常完善了。

 

[boud] 对 [胡泰斌经理] >> 听说庐山那本地人可以免费上去啊？ 是不是真的啊？

[胡泰斌经理] 对 [boud] >> 恩，是的。但是五一可能也会对本地人有一些限制。

 

[boud] 对 [胡泰斌经理] >> 你去过庐山吗？

[胡泰斌经理] 对 [boud] >> 去过啊，当然去过的。
